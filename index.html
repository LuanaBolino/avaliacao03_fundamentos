Principais comandos do GIt

git version

Mostra a versão do GIT

git --global user.name

Permite colocar o nome de usuário

git --global user.email

Usado para adicionar o email da conta

git config --global init.defaultBranch main

Configura o Git para usar o nome "main" como o nome padrão do branch principal quando um novo repositório é criado.

git config --global --unset user.email

Remove a configuração global do email do usuário no Git.

git init

Inicia um repositório em um diretório.

git status

Verifica o estado atual do repositório Git.

git add

Adiciona arquivos ao índice do Git, também conhecido como "staging area".

git commit nomedoarquivo -m

Cria um novo commit no Git. Um commit é um registro permanente das alterações feitas em um repositório em um determinado momento.

"A repetição é a chave para o aprendizado."
